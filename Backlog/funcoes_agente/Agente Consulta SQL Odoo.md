O VIIAH precisa ter integração com o a base de dados do Odoo para consultar os produtos

Resumo para o desenvolvedor:

Precisamos implementar uma consulta genérica de produtos (e serviços) no Odoo, que possa ser reutilizada por diferentes clientes, e não apenas pelo cliente Bira. A consulta atual é muito específica, então queremos uma versão mais ampla, com filtros por atributos comuns como cor, detalhe e características. A ideia é tornar essa busca flexível o suficiente para atender múltiplos clientes com necessidades variadas, sem depender de customizações específicas por cliente.
